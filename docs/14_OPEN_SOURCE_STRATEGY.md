# 14 — Open Source Strategy

## Positioning

Codex PM is a Codex-native delivery manager, not a generic task manager.

## README promise

Use cautious language:

```text
Codex PM helps Codex make verified progress from project docs.
```

Avoid:

```text
One-click complete any project.
```

## Differentiation

- docs-first
- Codex-native
- PM cerebellum architecture
- risk and verification gates
- execution memory
- evolution experiment mode

## Target community

- Codex users
- AI-native solo developers
- agent researchers
- docs-first builders
- open-source maintainers

## Contribution areas

- task parsers
- PM profiles
- memory algorithms
- risk rules
- prompt templates
- experiment reports
- examples

MIT is recommended for early adoption.
